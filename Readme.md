# TableExpressServer

This Server used for the ConnectivityPC App. The App Just Show up the some Data from ur Server. The Server can be ur PC or something u want.

## How to Install

```
git clone https://github.com/Tobias3107/tabletExpressServer.git

cd tabletExpressServer && npm i
```

Starting App 

```
npm start
```
